# Kahkahalar.com - 90'lar Web Deneyimi

> Web sitesi geliştirmeyi yeni öğrenmeye başlayanlar için yalnızca HTML kullanılarak yapılmış nostaljik bir site!

Bu proje, 90'lı yılların sonundaki internet sitelerinin sadeliğini yansıtan, tamamen temel HTML etiketleri kullanılarak oluşturulmuş bir web deneyimidir.

**Aynı zamanda içerisinde hiç CSS stillendirmesi olmadığından dolayı İĞRENÇ!** 😅

## 🚀 Projenin Amacı

Bu deponun temel amacı, web geliştirmeye yeni başlayanların CSS öğrenirken pratik yapabilecekleri bir temel sunmaktır.
Bu "kötü" siteyi alıp modern ve şık bir projeye dönüştürmek tamamen sizin elinizde!

Projeyi fork'layarak:
- CSS seçicileri (selectors) ile elemanları hedeflemeyi öğrenebilir,
- Flexbox veya Grid gibi modern yerleşim (layout) tekniklerini uygulayabilir,
- Renk paletleri, fontlar ve boşluklar (spacing) ile tasarıma estetik bir görünüm kazandırabilirsiniz.

---

## ☁️ Yayınlama (Deployment)

Bu GitHub reposu, `www.kahkahalar.com` özel alan adı ile Vercel kullanılarak yayınlanmaktadır.
Bu projeyi fork'layarak kendi alıştırmalarınız için kullanabilir, CSS ile stil kazandırarak daha iyi bir hale getirebilir hem de HTML'i tekrar etmiş olursunuz.

---

### 📂 Site İçeriği

Sitede aşağıdaki temel bölümler bulunmaktadır:
- Ana Sayfa
- Hakkında

- Fıkralar
- Komik Resimler

### VERCEL.COM - GİTHUB.COM - VS CODE birlikte kullanımı ile projelerinizi ister SUBDOMAİN isterseniz CUSTOMDOMAİN ile kendi alan adınız üzerinden eğer STATİC WEB SİTESİ ise ÜCRETSİZ yayınlayabilirsiniz.

###Bunun için sırasıyla :
1.VS CODE(IDE) u GİTHUB' a entegre edin ,
2.VERCEL.COM a girip GİTHUB hesabınızla giriş yapın üye olun.
3.GİTHUB ayarlarından (güvenlik için girhub mobile appınıza gelen kodu girerek) VERCEL'E GİTHUB REPOLARINIZA ERİŞİM YETKİSİ VERİN,
4.Github.com sayfasında sağüstte yer alan artıya tıklayarak daha sonra Create a New Repository tıklayın,
5.'Repository Name' kısmına projenize isim verin ve Create repository tıklayın,
6.Repositoryniz oluştuktan sonra ekranda gördüğünüz git komutlarından : branch, remote, push içeren son üç satırı seçerek kopyalayın.
Örneğin:
git branch -M main
git remote add origin https://github.com/tevfikg/test.git
git push -u origin main
7.Projenizi tamamladığınız VS CODE a gelip yeni terminal açın ve proje directorysinde olduğunuza emin olduktan sonra sırasıyla;
>git init (enter),
>git add .   (enter),
>6.adımdaki kopyalamış olduğunuz 3 komutu birden yapıştırın (enter),
8.tarayıcıda açık olan VERCEL hesabınıza girin ve github repolarınızdan projenizi bulun ve seçin daha sonra vercel.app uzantılı bir SUBDOMAİN ile ÜCRETSİZ yayınlayabilir,
isterseniz de kendi CUSTOM DOMAİN'inizden yayınlayabilrsiniz.
SSL sertifikanızı da VERCEL 90gün ÜCRETİSİZ Sağlayacaktır. 

Bu projeyi CSS ile stillendirip kendi eklemek istediklerinizle zenginleştirip yayınlayabilirsiniz.
